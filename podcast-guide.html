Pro Tips

  • Filter order matters! Keep them in this exact order
  • Adjust Noise Gate thresholds based on your room (noisy = lower, quiet = higher)
  • Always test with a 30-second recording before your actual podcast
  • Watch your audio meters - aim for green/yellow, avoid red!

Podcast Workflow

Pre-Recording

  • Connect Q802USB and test mic levels
  • Open OBS and check audio filters are enabled
  • Do 30-second test recording
  • Listen to test and adjust levels if needed
  • Prepare notes and talking points

Recording

  • Click "Start Recording" in OBS
  • Monitor audio levels (green/yellow = good, red = bad)
  • Maintain consistent mic distance (6-8 inches)
  • Don't worry about mistakes - edit later!
  • Click "Stop Recording" when done

Post-Production

  • Import recording into Audacity or Descript
  • Remove long pauses and mistakes
  • Add intro/outro music if desired
  • Normalize audio levels
  • Export as MP3 and publish!

Recording Tips

  • Two-Host Setup: Use two mics (both to Q802USB channels) for you and your co-host. Pan one left, one right for stereo separation.

Troubleshooting

  • Try different USB ports (prefer USB 2.0/3.0 on back panel)
  • Try a different USB cable
  • Wait 1-2 minutes for Windows to install drivers
  • Restart computer with Q802USB connected
  • Check Device Manager for "USB Audio CODEC"
  • Check Windows default recording device is set to "USB Audio CODEC"
  • In OBS Settings â†’ Audio, set "Mic/Auxiliary Audio" to Q802USB
  • Check Q802USB mixer: GAIN up, channel fader up, MAIN MIX up
  • Make sure mute button is NOT pressed on mixer
  • Speak loudly into mic and watch OBS audio meter
  • Lower GAIN on Q802USB mixer
  • Lower MAIN MIX fader
  • In OBS, audio meter should NOT hit red
  • Aim for green with occasional yellow peaks
  • Move microphone further from mouth (8-10 inches)
  • Increase GAIN on Q802USB mixer
  • Raise channel fader to near 0 dB
  • Raise MAIN MIX fader
  • Check XLR cable is firmly connected to mic and mixer
  • Move microphone closer (6 inches from mouth)
  • In OBS Compressor, increase Output Gain to +5 dB
  • Make sure Noise Suppression filter is enabled in OBS
  • Adjust Noise Gate threshold (lower = more aggressive)
  • Lower GAIN on unused mixer channels
  • Record in a quieter room or use sound dampening
  • Position mic correctly - dynamic mics need close placement (6-8 inches)
