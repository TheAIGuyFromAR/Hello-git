Scenario 1: Watch Party Reaction Video

Recording live reactions while watching a movie

Equipment Layout

Visual Setup

  • Logitech webcam on tripod - positioned to capture both hosts
  • 14" ring light - front and center, eye level
  • Monitor displaying Plex movie
  • Hosts seated side-by-side facing camera

Audio Setup

  • 2x XLR mics on boom arms (one per host)
  • Both mics → Q802USB Channels 1 & 2
  • Channel 1: Pan 30% LEFT
  • Channel 2: Pan 30% RIGHT

OBS Configuration (Step-by-Step)

1

Create New Scene

OBS: Scenes panel (bottom left) → Click "+"

Name it: Watch Party - Movie Title

2

Add Movie Source (Background Layer)

Sources panel → "+" → Window Capture

  • • Name: "Plex Movie"
  • • Window: Select [plex.exe]: Movie Title
  • • Capture Method: Windows 10/11 (best performance)
  • • Click OK
Make sure Plex is in fullscreen mode for clean capture
3

Add Webcam (Overlay Layer)

Sources panel → "+" → Video Capture Device

  • • Name: "Logitech Webcam"
  • • Device: Select your Logitech webcam
  • • Resolution: 1920x1080
  • • FPS: 30
  • • Click OK
4

Position Webcam (Picture-in-Picture)

Right-click webcam source → Transform → Edit Transform

Option A: Bottom Right Corner

  • • Position X: 1280
  • • Position Y: 680
  • • Size: 640x360

Option B: Bottom Left Corner

  • • Position X: 0
  • • Position Y: 680
  • • Size: 640x360
5

Add Border to Webcam (Optional but Recommended)

Right-click webcam → Filters → "+" → Image Mask/Blend

Or add a Color Source behind webcam:

  • • Sources → "+" → Color Source
  • • Name: "Webcam Border"
  • • Color: White or Purple (#667eea)
  • • Size: 650x370 (10px larger than webcam)
  • • Position behind webcam in source list
6

Configure Audio Levels

Audio Mixer (bottom of OBS)

  • Mic/Aux (Q802USB):
    • • Volume: 0 dB (or adjust to green/yellow)
    • • This is your voices - should be LOUDEST
  • Desktop Audio (Movie):
    • • Volume: -15 dB to -20 dB
    • • Movie audio should be background, not overpowering
Test levels BEFORE recording! Speak at normal volume and watch meters.
7

Add Branding (Optional)

Sources → "+" → Image

  • • Add podcast logo in top corner
  • • Add episode title as text overlay
  • • Position above all other sources

Pro Recording Tips for Watch Parties

  • Start recording BEFORE starting the movie
  • Record 10 seconds of silence at the beginning (for noise gate calibration)
  • Keep commentary natural - react authentically!
  • Pause movie to discuss longer thoughts (edit pauses out later)
  • Use headphones to monitor if movie audio is too loud
  • Have notes visible off-camera for discussion points

Scenario 2: Talking Head Review with Green Screen

Solo movie review with custom background

Equipment Layout

Visual Setup

  • Green sheet hung on wall OR white wall
  • Logitech webcam on tripod - eye level
  • 14" ring light - directly in front, slightly above
  • Host seated 3-4 feet from background

Audio Setup

  • 1x XLR mic on boom arm
  • Mic → Q802USB Channel 1
  • Pan: CENTER
  • Position mic 6-8 inches from mouth

OBS Configuration with Chroma Key

1

Create Scene & Add Background

Create new scene: "Talking Head Review"

Sources → "+" → Image

  • • Browse for background image (movie poster, cinema graphic, etc.)
  • • Right-click image → Transform → Fit to Screen
Background ideas: Movie posters, film reels, cinema seats, bokeh lights
2

Add Webcam Source

Sources → "+" → Video Capture Device

  • • Name: "Logitech Webcam - Host"
  • • Device: Select Logitech webcam
  • • Resolution: 1920x1080
  • • FPS: 30
3

Apply Chroma Key Filter

Right-click webcam source → Filters → "+" → Chroma Key

For Green Screen:

  • • Key Color Type: Green
  • • Similarity: 400-500
  • • Smoothness: 80-120
  • • Key Color Spill Reduction: 100

For White Wall:

  • • Use "Color Key" instead
  • • Click eyedropper, select wall color
  • • Similarity: 200-300
  • • Smoothness: 50-80
Common Issue: If you see green edges (spill), increase "Key Color Spill Reduction"
4

Fine-Tune Webcam Position

  • Centered: Position host in center of frame (rule of thirds)
  • Headroom: Leave some space above head (not cutting off top)
  • Composition: Host should take up ~60-70% of frame
5

Add Lower Third (Name/Title)

Sources → "+" → Text (GDI+)

  • • Text: "Your Name | Film Critic"
  • • Font: Bold, size 48-60
  • • Color: White with black outline
  • • Position: Bottom left corner

Green Screen Lighting Tips

GOOD Lighting:

  • Even light across entire green screen
  • No shadows on background
  • Ring light creates nice catchlights in eyes
  • Separate lighting for host and background

BAD Lighting:

  • Harsh shadows behind host
  • Wrinkles/creases in green screen
  • Green spill/reflection on host's skin
  • Uneven lighting causing dark spots

Quick Fix: Stand 3-4 feet away from green screen to minimize spill. If using white wall, make sure it's evenly painted!
